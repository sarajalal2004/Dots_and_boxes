### ***Description***
#### A website summulating the common dots and boxes game depending on HTML, CSS, JS
#### Each player tries to complete boxes while blocking their opponent from doing the same.
***
### ***Technologies Used***
* VS code
    * HTML
    * CSS
    * JavaScript
* gitHub / gitBash
